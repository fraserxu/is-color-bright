# is-color-bright
is-color-bright


It takes in a rgb value like `rgb(255, 255, 255)`, it returns a boolean value of whether is bright or not

## Installation

```
$ npm install is-color-bright
```

## Usage

```js
const isColorBright = require('is-color-bright')

// return true/flase
isColorBright('rgb(255, 255, 255)')
```
